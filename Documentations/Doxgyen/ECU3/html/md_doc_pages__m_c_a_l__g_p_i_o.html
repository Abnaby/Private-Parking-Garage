Notes

  • Enabling the alternate function of some of the port pins does not affect the use of the other pins in the port as general digital I/O.

V1 Features :

Can Control I/O Controller Pins As :
  1 - Set Direction Of Pin I/O.
  2 - Set Output Value Of Pin even High or Low.
  3 - Read Value Of PIN If it Input.
  4 - Toggle PIN in Specific Port.
  5 - Set Output Value Of All PINS in Port even High or Low.
  6 - Set Direction Of All PINS in Port.
