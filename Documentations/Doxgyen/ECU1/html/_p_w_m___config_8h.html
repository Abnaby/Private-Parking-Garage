#define TIM1_ANTI_GLITCH   ANTI_GLITCH_DISABLE

This Macro Used to EN/Disable Anti Glitch For Timer1 Only.

If ANTI-GLITCH is Enabled Then OCR1A(PD5) is Researved For Buffering If ANTI-GLITCH is Disabled you have freedom to use OCR1B(PD4)
